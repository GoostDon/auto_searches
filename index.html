Auto Random Searches

Here you can open random searches in your chosen search engine.(for informational purposes only)










You need to allow pop-ups for this webpage, or else this website may not work correctly!

How it works:
1. Make sure you meet the requirements(see below)
2. Please don't block the ads on this website(I'm sorry, hosting a site is not free...)
3. Select your search engine
4. (optional) Change the number of searches and the delay in seconds
5. Press start or hit "Enter"
6. Random searches will start in new tabs with your chosen delay between the searches

Requirements:
- Read and be sure to understand the Disclaimer before you use this website(see below)
- Allow pop-ups and redirections (for this website) in your browser
- Enable JavaScript (for this website)
- Make sure you can connect to your chosen search engine
- Use a recent browser (Edge, Chrome, Firefox, etc. should be fine. If it doesn't work, you are probably using an old browser and really should update!)
